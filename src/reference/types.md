# الأنواع المدمجة

تُقدّم لغة ص **تسعة أنواع مدمجة**. أسماؤها يُصدرها المعجمي **مُعرِّفات** (لا كلمات
محجوزة)، فيجوز استعمالها أسماءَ متغيّرات خارج موضع النوع.

> **المصدر:** [`language-truth/keywords.yaml`](https://github.com/sadlang/s-programming-language/blob/sadlang/language-truth/keywords.yaml)
> (فئة `builtin_types`) و[`language-truth/types.yaml`](https://github.com/sadlang/s-programming-language/blob/sadlang/language-truth/types.yaml).

| النوع | الإنجليزيّة | الفئة | الوصف |
|------|-------------|------|-------|
| `رقم` | integer | عدديّ | عدد صحيح |
| `عشري` | double | عدديّ | عدد عشريّ بفاصلة عائمة |
| `نص` | string | نصّيّ | سلسلة نصّيّة UTF-8 |
| `منطقي` | boolean | منطقيّ | `صحيح` أو `خطأ` |
| `فراغ` | void | خاصّ | لا قيمة (مردود دالة لا تُرجع) |
| `عدم` | null | خاصّ | النوع الفارغ المتمايز |
| `مصفوفة` | array | مركّب | قائمة عناصر مرتّبة |
| `خريطة` | map | مركّب | أزواج مفتاح/قيمة |
| `أي` | any | خاصّ | أيّ نوع (ديناميكيّ) |

## أمثلة

```sad
متغير العدد: رقم = 42
متغير النسبة: عشري = 3.14
متغير الاسم: نص = "مرحبًا"
متغير مفعّل: منطقي = صحيح
متغير الأرقام: مصفوفة = [1، 2، 3]
متغير المعجم: خريطة = {"أ": 1، "ب": 2}
```

## الأنواع الاختياريّة وأمان العدم

يُكتب النوع الاختياريّ باللاحقة `؟` (العربيّة، U+061F) ليقبل القيمة أو `لاشيء`:

```sad
متغير الوسط: نص؟ = لاشيء
```

يُعالَج بعوامل أمان العدم: `؟.` (وصول آمن)، `؟؟` (اندماج فارغ)، `مؤكد` (تأكيد عدم
الفراغ). راجع [العوامل والأسبقيّة](operators.md#أمان-العدم-عربيّ-حصرًا).

## ملاحظة تصميميّة

لا يوجد نوع باسم `ANY` على مستوى محرّك القيم الداخليّ؛ نوع `أي` (TYPE_ANY) موجّه
للمستخدم، بينما القيم تُمثَّل داخليًّا بنوع `Value` موحّد. هذا تفصيل تنفيذيّ يخصّ
[دليل المطوّرين الداخليّ](https://github.com/sadlang/dev-guide) ولا يؤثّر في كتابة البرامج.
