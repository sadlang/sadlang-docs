# مصدر الحقيقة الموحّد (SoT)

تتبنّى لغة ص مبدأ **مصدر الحقيقة الموحّد** (Single Source of Truth): تُوصَف اللغة —
كلماتها وأنواعها وعواملها وقواعدها وأخطاؤها — في ملفّات بيانات مُحكَّمة بمخطّطات، ثم
يُولَّد منها الكود والتوثيق. هذا يمنع التضارب بين الوصف والسلوك.

> **الموقع:** [`language-truth/`](https://github.com/sadlang/s-programming-language/tree/sadlang/language-truth)
> في المستودع الأساسيّ.

## لماذا؟

- **اتّساق:** تعريف واحد للكلمة/النوع يُغذّي المعجمي والمحلّل والتوثيق معًا.
- **تحقّق آليّ:** الـCI يطابق الوصف بالسلوك (حقول `maps_to` و`conformance`).
- **توثيق حيّ:** هذا الكتاب وصفحات `docs/parser_rule/_generated` تُشتقّ من المصدر.

## أبرز الملفّات

| الملفّ | يحوي |
|-------|------|
| `keywords.yaml` | المعجم: 40 محجوزة + 3 عوامل + سياقيّة + 9 أنواع |
| `operators.yaml` | العوامل وأسبقيّتها وترابطها |
| `types.yaml` | الأنواع المدمجة وخصائصها |
| `grammar/*.yaml` | قواعد الإنتاج (EBNF) عبر ثماني طبقات |
| `errors/` | كتالوج رموز الأخطاء ورسائلها |
| `builtins/` | الدوال المضمّنة |
| `stdlib/` | وحدات المكتبة القياسيّة |
| `_schemas/` | مخطّطات JSON التي تُحكِّم كلّ ما سبق |

## القاعدة الذهبيّة

عند أيّ تعارض بين هذا التوثيق و`language-truth/`، فإنّ **`language-truth/` هو الفيصل**.
دور هذا المستودع أن يَعرض الحقيقة ويشرحها للقارئ، لا أن يُنشئ قاعدةً جديدة.
