# أوّل برنامج: أهلًا يا عالم

لنكتب أبسط برنامج ونشغّله، ثم نضيف متغيّرات وتحكّمًا بالتدفّق لنرى ملامح اللغة.

## الملفّ

أنشئ ملفًّا باسم `أهلا.ص` بالمحتوى التالي:

```sad
دالة رئيسية()
    اطبع("أهلًا يا عالم")
نهاية
```

ثم شغّله بالمفسّر:

```powershell
.\build\bin\Debug\sad.exe أهلا.ص
```

المخرجات:

```text
أهلًا يا عالم
```

## شرح سطرًا سطرًا

- `دالة رئيسية()` — تعريف دالة الدخول `رئيسية` (نقطة بدء البرنامج).
- `اطبع(...)` — دالة مضمّنة تطبع نصًّا على المخرج القياسيّ.
- `نهاية` — إغلاق كتلة الدالة. **كلّ كتلة في لغة ص تُغلَق بـ`نهاية`**.

## مثال أوسع

```sad
دالة رئيسية()
    متغير الاسم = "أحمد"
    متغير العمر = 30

    اطبع("الاسم: " + الاسم)

    إذا العمر >= 18
        اطبع("بالغ")
    وإلا
        اطبع("قاصر")
    نهاية

    لكل رقم في [1، 2، 3]
        اطبع(رقم)
    نهاية
نهاية
```

لاحظ:

- `متغير` يُعرِّف متغيّرًا قابلًا للتغيير (و`ثابت` يُعرِّف ثابتًا).
- الفاصلة `،` العربيّة تكافئ `,` تمامًا داخل عناصر المصفوفة.
- `إذا/وإلا` و`لكل ... في` كلٌّ منهما كتلة تُغلَق بـ`نهاية`.
