# معالجة الأخطاء

تتبع لغة ص نموذج «حاول/امسك/أخيرًا» مع كلمة `ارمي` لإطلاق الأخطاء.

## حاول / امسك / أخيرًا

```sad
حاول
    متغير الناتج = اقسم(10، 0)
    اطبع(الناتج)
امسك خطأ
    اطبع("حدث خطأ: " + خطأ.الرسالة)
أخيراً
    اطبع("انتهى التنفيذ")
نهاية
```

- `حاول` يفتح الكتلة المحميّة.
- `امسك` يلتقط الخطأ (مع ربطه باسم).
- `أخيراً` يُنفَّذ دائمًا (نجح أو فشل).

## إطلاق خطأ

```sad
دالة اقسم(أ، ب)
    إذا ب == 0
        ارمي "القسمة على صفر غير مسموحة"
    نهاية
    ارجع أ / ب
نهاية
```

الكلمات المحجوزة: `حاول`، `امسك`، `ارمي`، `أخيراً`. راجع
[الكلمات المحجوزة الأربعون](../reference/keywords.md).
