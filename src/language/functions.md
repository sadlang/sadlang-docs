# الدوال

تُعرَّف الدوال بكلمة `دالة`، وتُعيد القيم بـ`ارجع`. دالة الدخول للبرنامج اسمها `رئيسية`.

## التعريف والاستدعاء

```sad
دالة اجمع(أ، ب)
    ارجع أ + ب
نهاية

دالة رئيسية()
    متغير الناتج = اجمع(3، 4)
    اطبع(الناتج)        # 7
نهاية
```

## التصريح بالأنواع والمردود

```sad
دالة مربّع(س: رقم): رقم
    ارجع س * س
نهاية
```

## دوال لامدا (تعبيريّة)

كلمة `لامدا` سياقيّة تُنشئ دالة مجهولة الاسم:

```sad
متغير ضاعِف = لامدا(س) ارجع س * 2 نهاية
اطبع(ضاعِف(5))          # 10
```

## المولّدات و`أنتج`

```sad
دالة أعداد()
    أنتج 1
    أنتج 2
    أنتج 3
نهاية
```

الكلمات ذات الصلة: `دالة`، `ارجع` (محجوزتان)، و`لامدا`، `أنتج`، `مولد`، `ترجع`
(سياقيّة). راجع [الكلمات المحجوزة الأربعون](../reference/keywords.md).
