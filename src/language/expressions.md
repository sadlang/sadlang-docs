# التعبيرات والعمليات

تدعم لغة ص مجموعة كاملة من العوامل الحسابيّة والمنطقيّة والمقارنة، مع ترتيب أسبقيّة
محدّد بدقّة. هذا الفصل لمحة؛ والجدول الكامل في [العوامل والأسبقيّة](../reference/operators.md).

## الحسابيّة

```sad
متغير المجموع = 2 + 3 * 4      # 14 (الضرب أسبق)
متغير الباقي = 17 % 5          # 2
متغير القوّة = (2 + 3) * 4      # 20 (الأقواس تُقدَّم)
```

## المقارنة والمنطق

العوامل المنطقيّة الثلاثة **كلماتٌ عربيّة**: `و` (and)، `أو` (or)، `ليس` (not).

```sad
إذا العمر >= 18 و العمر < 65
    اطبع("ضمن سنّ العمل")
نهاية

إذا ليس مكتمل
    اطبع("غير مكتمل")
نهاية
```

## العامل الثلاثيّ

اختصارٌ للشرط داخل تعبير (`شرط ؟ قيمة_صحّة : قيمة_خطأ`). رمز السؤال يختلف بحسب
الإصدار — راجع [جدول العوامل](../reference/operators.md) لرمز إصدارك الدقيق:

```sad
متغير الإشارة = س > 0 ؟ 1 : -1
```

## دمج النصوص

```sad
متغير التحيّة = "مرحبًا يا " + الاسم
```

راجع [العوامل والأسبقيّة](../reference/operators.md) لكلّ العوامل ودرجات أسبقيّتها (1 = الأعلى).
