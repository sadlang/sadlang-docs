# المتغيّرات والثوابت

تُعرَّف المتغيّرات في لغة ص بكلمة `متغير`، والثوابت بكلمة `ثابت`. تُستنتَج الأنواع
تلقائيًّا من القيمة، ويمكن التصريح بالنوع صراحةً عند الحاجة.

## التعريف

```sad
متغير العدّاد = 0          # متغيّر قابل للتغيير
ثابت باي = 3.14159        # ثابت لا يتغيّر

العدّاد = العدّاد + 1      # إعادة إسناد مسموحة للمتغيّر
# باي = 3                 # خطأ: لا يمكن تغيير الثابت
```

## التصريح بالنوع

أسماء الأنواع (مثل `رقم`، `نص`، `عشري`) **ليست محجوزة**، وتُكتب بعد الاسم عند الرغبة
في التصريح الصريح:

```sad
متغير العمر: رقم = 30
متغير الاسم: نص = "سارة"
```

## محدّدات إضافيّة

- `ساكن` — عضو/متغيّر ساكن مرتبط بالصنف لا بالنسخة.
- `خارجي` — رمز معرَّف خارج الوحدة (ربط خارجيّ).
