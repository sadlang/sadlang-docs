# الوحدات والاستيراد

تُنظَّم البرامج في وحدات، مع كلمات `استورد`، `من`، `كـ`، `صدّر`.

## الاستيراد

```sad
استورد "رياضيات"                  # استيراد وحدة كاملة
استورد جذر، جا من "رياضيات"        # استيراد رموز محدّدة
استورد "رياضيات" كـ ر             # استيراد باسم بديل
```

## التصدير

```sad
صدّر دالة مساحة_دائرة(نق)
    ارجع 3.14159 * نق * نق
نهاية
```

## الكلمات

| الكلمة | الإنجليزيّة | الدور |
|--------|-------------|------|
| `استورد` | import | جلب وحدة/رموز |
| `من` | from | تحديد مصدر الرموز |
| `كـ` | as | اسم بديل |
| `صدّر` | export | إتاحة رمز للوحدات الأخرى |

تشمل المكتبة القياسيّة وحدات `core`، `io`، `math`، `string`، `network`، `graphics`.
راجع [الكلمات المحجوزة الأربعون](../reference/keywords.md).
