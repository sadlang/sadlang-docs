# التحكّم بالتدفّق

تُوفّر لغة ص الشروط والحلقات ومطابقة الأنماط. كلّ كتلة تُغلَق بـ`نهاية`.

## الشرط `إذا / وإلا`

```sad
إذا الدرجة >= 90
    اطبع("ممتاز")
وإلا
    اطبع("دون الممتاز")
نهاية
```

## الحلقة `بينما`

```sad
متغير العدّاد = 0
بينما العدّاد < 5
    اطبع(العدّاد)
    العدّاد = العدّاد + 1
نهاية
```

## الحلقة `لكل ... في`

```sad
لكل عنصر في ["أ"، "ب"، "ج"]
    اطبع(عنصر)
نهاية
```

تتحكّم `توقف` (break) و`استمر` (continue) في سير الحلقة.

## مطابقة الأنماط `طابق`

```sad
طابق الحالة
    عندما 1
        اطبع("واحد")
    عندما 2
        اطبع("اثنان")
    افتراضي
        اطبع("غير ذلك")
نهاية
```

الكلمات: `طابق` (فتح)، `عندما` (حالة)، `افتراضي` (الحالة الافتراضيّة). راجع
[الكلمات المحجوزة الأربعون](../reference/keywords.md) لكامل كلمات التحكّم.
