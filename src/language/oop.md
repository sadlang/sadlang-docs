# البرمجة الكائنيّة

تدعم لغة ص الأصناف والبنى والتعدادات والوراثة والسمات وتحميل العوامل الزائد.

## الأصناف

```sad
صنف شخص
    خاص الاسم: نص
    خاص العمر: رقم

    باني(الاسم، العمر)
        هذا.الاسم = الاسم
        هذا.العمر = العمر
    نهاية

    عام دالة تحيّة()
        ارجع "أنا " + هذا.الاسم
    نهاية
نهاية

دالة رئيسية()
    متغير ص = جديد شخص("ليلى"، 25)
    اطبع(ص.تحيّة())
نهاية
```

- `صنف` يفتح الصنف، `باني` المُنشئ، `هذا` يشير إلى النسخة الحاليّة.
- `جديد` يُنشئ نسخة.
- محدّدات الوصول: `عام`، `خاص`، `محمي`؛ و`مجرد` للصنف/العضو المجرَّد.

## الوراثة

```sad
صنف موظّف يرث شخص
    خاص الراتب: عشري

    باني(الاسم، العمر، الراتب)
        الأساس(الاسم، العمر)   # استدعاء باني الأب
        هذا.الراتب = الراتب
    نهاية
نهاية
```

`يرث` للوراثة، و`الأساس` للوصول إلى الصنف الأب.

## بنى وتعدادات

```sad
بنية نقطة
    س: رقم
    ص: رقم
نهاية

تعداد لون
    أحمر
    أخضر
    أزرق
نهاية
```

## الخصائص وتحميل العوامل

كلمات سياقيّة مثل `خاصية` و`احصل`/`عيّن` (getter/setter)، و`عامل` لتحميل العوامل.
راجع [الكلمات المحجوزة الأربعون](../reference/keywords.md) لكامل كلمات الكائنيّة.
