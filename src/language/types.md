# الأنواع والقيم الحرفيّة

تُقدّم لغة ص **9 أنواع مدمجة**. أسماؤها **ليست كلمات محجوزة** (يُصدرها المعجمي كمُعرِّفات)،
فيمكن استعمالها أسماءً خارج موضع النوع، لكنها تُفسَّر أنواعًا في موضع التصريح.

| النوع | الاسم الإنجليزيّ | المعنى | مثال |
|------|------------------|--------|------|
| `رقم` | integer | عدد صحيح | `42` |
| `عشري` | double | عدد عشريّ | `3.14` |
| `نص` | string | سلسلة نصّيّة | `"مرحبًا"` |
| `منطقي` | boolean | صحيح/خطأ | `صحيح`، `خطأ` |
| `فراغ` | void | لا قيمة (مردود دالة) | — |
| `عدم` | null | النوع الفارغ المتمايز | `لاشيء` |
| `مصفوفة` | array | قائمة مرتّبة | `[1، 2، 3]` |
| `خريطة` | map | أزواج مفتاح/قيمة | `{"أ": 1}` |
| `أي` | any | أيّ نوع | — |

## القيم الحرفيّة المحجوزة

ثلاث قيم حرفيّة هي **كلمات محجوزة**:

```sad
متغير يعمل = صحيح     # true
متغير متوقّف = خطأ    # false
متغير قيمة = لاشيء    # null
```

## أمان العدم (Null Safety)

تميّز لغة ص بين القيمة العاديّة والقيمة التي قد تكون «عدمًا». يُكتب النوع الاختياريّ
بلاحقةٍ على النوع (مثل `نص?`)، ويُعالَج بعامل **وصول آمن** وعامل **اندماج فارغ**.

> **الرمز يختلف بحسب الإصدار:** الرموز الدقيقة لأمان العدم مُولَّدة من مصدر الحقيقة
> لكلّ قناة — راجع [العوامل والأسبقيّة](../reference/operators.md) و
> [الأنواع المدمجة](../reference/types.md) لرؤية رموز إصدارك بالضبط (مع أمثلة جاهزة).
